Quick Start

From zero to mining
in four steps.

01
Install the SDK

Install the NeuralChain Python SDK via pip. Requires Python 3.9+ and CUDA for GPU mining subnets.

pip install bittensor
# Verify installation
python -c "import bittensor; print(bt.__version__)"
02
Create Your Wallet

Generate a coldkey (your long-term identity) and a hotkey (used for signing transactions). Store your mnemonics safely — they cannot be recovered.

btcli wallet create --wallet.name mykey
btcli wallet new_hotkey --wallet.name mykey
03
Choose a Subnet

Browse the active subnets and pick one that matches your hardware and expertise. Each subnet has different requirements and reward structures.

import bittensor as bt
sub = bt.subtensor('finney')
# List all active subnets
sub.get_all_subnets_info()
04
Register & Mine

Register on your chosen subnet, deploy your miner script, and connect to the network. The system will begin scoring your outputs and distributing TAO rewards immediately.

# Register (costs a small TAO fee)
sub.register(wallet=wallet,netuid=19)
# Deploy miner
python neurons/miner.py \
  --wallet.name mykey \
  --subtensor.network finney

Examples

Real code for real use cases.

Language Models
Run an LLM Miner (SN-19)

Deploy a Llama or Mistral model as a miner on the OpenTAO language subnet. Respond to text generation queries and earn TAO.

# miner.py — LLM subnet
from transformers import pipeline
import bittensor as bt

model = pipeline("text-generation",
  model="meta-llama/Llama-3")

# Axon handles incoming requests
axon = bt.axon(wallet=wallet)
axon.attach(forward_fn=model)
axon.serve(netuid=19)
Validation
Write a Custom Validator

Score miners' outputs using your own logic and submit weights to the consensus mechanism. Earn TAO for accurate, consistent scoring.

# validator.py
import bittensor as bt

async def score_outputs(responses):
  scores = []
  for r in responses:
    score = evaluate_quality(r)
    scores.append(score)
  return scores

# Submit weights on-chain
sub.set_weights(netuid=19,
  weights=scores)
Subnet Creation
Launch Your Own Subnet

Define a new AI task, design incentive structures, and register your subnet on the NeuralChain network. Bond TAO to secure your slot.

# register_subnet.py
import bittensor as bt

sub = bt.subtensor('finney')

# Register new subnet
sub.register_subnetwork(
  wallet=wallet,
  wait_for_inclusion=True
)

# Your subnet is live! 🎉
